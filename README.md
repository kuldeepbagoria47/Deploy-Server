# MLE Website

This repos has folder & files for mle website backend node server




## Tech Stack


**Server:** Node, Express
**Database:** MongoDB




## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
